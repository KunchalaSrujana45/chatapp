Project Description

This project is a Real-Time Chat Application developed using Spring Boot and WebSocket.
It allows users to send and receive messages instantly without refreshing the page.

The main goal of this project is to demonstrate real-time communication between multiple users using WebSocket technology.

 Objectives

To implement real-time messaging

To understand WebSocket communication

To build a responsive chat system

To enable multiple users to interact simultaneously

Technologies Used

Java

Spring Boot

WebSocket (STOMP Protocol)

HTML, CSS, JavaScript

Maven



No page refresh required

Simple and user-friendly interface
